Для тестировщиков в Veeam важны технические знания, поэтому перед участием в финальном туре на позиция ручного тестировщика предлагаем Вам изучить несколько вопросов по основам сетей и файловых систем.

Для этого нужно сделать следующее:
1) Найти в бумажном или электронном виде книгу по администрированию Windows Server 2008/2012/2016.
2) Скачать с сайта www.vmware.com продукт VMware Workstation в evaluation/trial режиме.
3) Установить 2-3 виртуальные машины с Windows Server (2008/2012/2016) и Windows Client OS (Windows 7/8/10).
4) Проработать несколько вопросов:

Настройки сети (IP, mask, default gateway, DNS) Зачем нужна каждая из них, когда и для чего используется.
DNS: назначение, принцип работы, типы зон, различные типы записей и их назначение.
DHCP: назначение, принцип работы, какие настройки может раздавать клиентам, резервирование адресов.
Что можно делать с клиентской машины при взаимодействии с DNS и DHCP.
Файловая система NTFS: основные отличия от FAT32, возможности (отказоустойчивость, сжатие, шифрование, журналирование, безопасность и т.д.).
 
При изучении каждого вопроса важно читать не только теорию, но и обязательно пробовать всё на практике: поднять домен-контроллер, добавить машину в домен, поднять DNS и DHCP, разобраться в их функции и настройках и т.д. Эффективность таких занятий на порядок выше.

В ходе работы важно понимать суть происходящего, цели тех или иных служб/компонентов, механизм их работы и взаимодействия.
